✂️ _[Visual Studio Code][vscode] snippet library for JavaScript, Markdown, and Python._

## JavaScript

| Prefix | Description                 |
| ------ | --------------------------- |
| `log`  | Logs output to the console. |
| `imp`  | Import statement.           |
| `nimp` | Named import statement.     |
| `af`   | Arrow function.             |
| `aaf`  | Anonymous arrow function.   |

## Jest

| Prefix | Description      |
| ------ | ---------------- |
| `jd`   | Jest describe.   |
| `jt`   | Jest test.       |
| `je`   | Jest expect.     |
| `jba`  | Jest beforeAll.  |
| `jaa`  | Jest afterAll.   |
| `jbe`  | Jest beforeEach. |
| `jae`  | Jest afterEach.  |

## React

## Markdown

| `mli`     | Inline style link.    |
| `mb`      | Bold text.            |
| `mi`      | Italic text.          |
| `mbq`     | Blockquote.           |
| `mh`      | First level header.   |
| `mhh`     | Second level header.  |
